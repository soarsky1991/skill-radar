# Contributing

谢谢你愿意帮 HyperFrames Creator Kit 变得更好。这个项目欢迎小贡献，因为教程仓库能长大，靠的往往不是一次性大重构，而是很多可验证的小改进。

## 最需要的贡献

- 增加一个可复现 lesson。
- 补充一个真实失败案例和修复方法。
- 增加一个平台发布 checklist。
- 补一个字幕样式、封面 brief 或 SRT 示例。
- 改善 README、FAQ、术语表和英文摘要。
- 把一个本地经验改写成不含私人信息的公开模板。

## Lesson 标准

每个 lesson 至少包含：

- Promise：学习者能做出什么。
- Inputs：需要哪些素材、数据或文件。
- Steps：按步骤能复现。
- Checks：如何证明跑通。
- Output：最后产物是什么。
- Risks：版权、隐私、平台、AI 标注和夸张承诺风险。

## Pull Request 检查

提交前请自查：

- 没有密钥、token、验证码、账号后台截图或临时签名 URL。
- 没有未授权名人、影视角色、品牌素材。
- 教学 claim 有录屏、截图、输出或检查支撑。
- 新文件路径和 README 链接能被读者找到。
- 中文主线清楚；英文可以先覆盖摘要和入口。

## 好的 first contribution

- 给 Lesson 01 增加一个真实示例输入表。
- 给 `templates/publishing-pack.md` 补一个平台风险项。
- 给 `GROWTH_PLAYBOOK.zh-CN.md` 补一个高星教程仓库拆解。
- 把某个术语解释成中英双语。

## 维护原则

- 先证明，再包装。
- 先 lesson，再百科。
- 先真实操作，再动效。
- 先小而可跑，再大而全。
