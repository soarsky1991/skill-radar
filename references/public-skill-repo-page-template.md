# Public Skill Repo Page Template

Use this template for future public repos that spin out of Agent Skill Radar.

## Above The Fold

- Name: clear, searchable, and memorable.
- One-line promise: what the repo helps a builder do.
- Proof line: the GitHub demand signal that justified the repo.
- Primary CTA: install, run, star, or open an issue with a missing case.

## README Structure

1. What problem this solves.
2. Who it is for.
3. A tiny demo or output sample.
4. Quick start.
5. Why this exists now, with links to public evidence.
6. Roadmap limited to the next two weeks.
7. Contribution path for missing repos, edge cases, and compatibility reports.

## Repository Metadata

Description formula:

```text
<Action> for <audience> using <evidence/source>.
```

Topic mix:

```text
ai-agents, agent-tools, developer-tools, mcp, codex, claude-code, github, open-source
```
